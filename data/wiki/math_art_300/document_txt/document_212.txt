Introduction to Wikipedia Wikipedia is made by people like you.

This page takes you through a set of tutorials aimed at complete newcomers interested in contributing.

It covers all the basics, and each tutorial takes only a few minutes, so you can become a proficient Wikipedian in no time! Get started Policies and Guidelines The wiki markup source editor shows the underlying page source code.

It works like a plain text file, indicating links and other items using simple code like this: [[Earth]].

Editing Referencing Images Tables VisualEditor is the easier way of editing that works more like a word processor and hides the underlying source code.

Links and other items are edited using toolbar and pop-up interfaces.
